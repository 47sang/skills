# stock-research — A股股票数据查询与投研报告生成

## 数据源
- **akshare**（东方财富、新浪财经等数据接口）
- 读取年报 PDF（由 `annual-report` skill 下载）

## 三个使用场景

### 场景A：查股价
用户问"XX股价多少"、"XX走势如何"
→ 直接获取实时数据，返回简要行情

### 场景B：查财报
用户问"XX年报怎么样"、"XX财务数据"
→ 获取财务指标，直接返回分析

### 场景C：生成投研报告
用户明确要求"生成研究报告"、"分析一下XX"
→ 获取数据 + 生成完整 Markdown 报告

## 核心脚本

### fetch_stock_data.py — 获取股票基本面数据
```bash
python3 /Users/zhou/.hermes/skills/openclaw-imports/stock-research/scripts/fetch_stock_data.py <股票代码>
```

输出：公司简介 + 核心财务数据 + 实时行情，同时保存 JSON 到 `/tmp/`

### read_pdf.py — 读取年报 PDF（来自 annual-report skill）
```bash
python3 /Users/zhou/.hermes/skills/openclaw-imports/annual-report/scripts/read_pdf.py <PDF路径> [起始页] [结束页] [关键词]
```
**注意**：直接使用系统 python3，无需激活虚拟环境。

## 报告生成流程

1. **获取公司基本信息**：名称、行业、主营业务、上市日期
2. **获取财务数据**：利润表、资产负债表、现金流量表
3. **获取实时行情**：股价、涨跌幅、市值、估值
4. **读取年报PDF**：从已下载的年报中提取业务描述、核心竞争力
5. **分析框架**：
   - 主营业务分析（收入结构、产品占比）
   - 核心财务指标（营收、净利、ROE、经营现金流）
   - 估值分析（PE、PB、PS）
   - 行业地位与竞争格局
   - 风险提示
6. **输出报告**：按命名规范保存

## 报告输出规范

### 命名格式
```
{股票名称}_{股票代码}_研究报告.md
{股票名称}_{股票代码}_交易策略.md
{股票名称}_{股票代码}_策略动态调整.md
```

### 保存位置
- 个股研报：`/Users/zhou/Desktop/投研分析/个股票投研报告/`
- 主题研报（行业/产业链）：`/Users/zhou/Desktop/投研分析/` 父目录

### 报告结构
```markdown
# {股票名称}（{股票代码}）投研报告

> **报告日期**：YYYY-MM-DD | **数据基准**：YYYY年度报告 + YYYY年Q1
> **风险等级**：⭐⭐⭐⭐☆ | **投资建议**：买入/持有/观望（带价格区间）

## 头部信息
- 股票代码、公司全称、行业、上市时间
- 当前股价、市值、估值水平

## 公司概况
- 主营业务概述（从年报提取，非网络搜索）
- 核心产品/服务
- 行业地位与竞争优势
- 实际控制人及股权结构

## 主营业务分析
- 收入结构（各业务线/产品占比）
- 毛利率分析
- 区域分布

## 财务分析
- 近3-5年核心财务指标对比表
- 盈利能力（毛利率、净利率、ROE）
- 成长性（营收增速、净利增速）
- 现金流质量（经营现金流/净利润）
- 偿债能力（资产负债率）

## 管理层讨论要点
- 年报中的管理层分析摘要
- 未来战略规划
- 风险提示

## 估值分析
- 当前估值水平（PE/PB/PS）
- 与同行业可比公司对比
- 历史估值区间

## 风险提示
- 行业风险
- 公司特定风险
- 宏观经济风险

## 投资建议
- 目标价区间
- 买入/持有/观望建议
- 关键跟踪指标
```

## 分析流程（四维框架）

1. **业务维度**：看主营业务、收入结构、毛利率
2. **财务维度**：看利润表、现金流、资产负债
3. **估值维度**：看PE/PB/PS与行业对比
4. **风险维度**：看高管减持、股东质押、商誉等

## 注意事项
- 优先从 `年报PDF/` 目录读取已下载的年报，而非网络搜索
- 财务数据以年报/季报为准，网络数据仅作验证
- 所有估值数据标注日期，避免用过时数据
- 引用传闻/自媒体数据须标注来源并交叉验证
